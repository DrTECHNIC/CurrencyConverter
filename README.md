# Конвертер валют

Простое настольное приложение для быстрой конвертации сумм между различными валютами.

## Системные требования

- Операционная система: Windows 10 или новее
- Дополнительное ПО не требуется

## Установка и запуск

1. Скачайте файл `currency_converter.exe` из репозитория проекта
2. Сохраните файл в удобную для вас папку
3. Запустите приложение двойным щелчком мыши по файлу `currency_converter.exe`

## Использование

1. В поле "Сумма" введите числовое значение для конвертации
2. В выпадающем списке "Из" выберите исходную валюту
3. В выпадающем списке "В" выберите целевую валюту
4. Нажмите кнопку "Конвертировать"
5. Результат конвертации отобразится в нижней части окна

## Поддерживаемые валюты

- RUB - Российский рубль
- USD - Доллар США
- EUR - Евро
- GBP - Фунт стерлингов
- IDR - Индонезийская рупия
- KZT - Казахстанский тенге

## Особенности

- Программа работает автономно, не требует подключения к интернету
- Интерфейс прост и интуитивно понятен
- Не требует установки дополнительных библиотек
